## Installation

1. Clone this repository.
2. Install dependencies using PNPM:

   ```bash
   pnpm install
   ```

3. Set up your environment variables (if needed).

## Usage

1. Start the development server:

   ```bash
   pnpm dev
   ```

2. Access the API at `http://localhost:5000`.
